Land surface Interactions with the Atmosphere over the Iberian Semi-arid Environment


Database

The LIAISE database is hosted by AERIS under the HyMeX data policy. The website is being finalized, but currently field campaign quick-look images have been posted. A basic catalogue and calendar is currently being prepared and will be continuously updated here. The SOP UAV Flight Summary contains detailed information on flights and measurements and will be formatted for the Table below.

Data can now be uploaded (including metadata) and downloaded from the LIAISE AERIS DB. A document describing the database usage procedures will be continuously updated (based on input from users).
